Endpoints
POST /scan Run full identity scan

Initiates a full OSINT scan against an email address or username. Returns a scan_id for async retrieval, or full results synchronously if sync: true.

// Request body
{
  "target": "john.doe@example.com",
  "type": "email", // "email" | "username" | "phone"
  "depth": "deep", // "quick" | "standard" | "deep"
  "sync": false, // true returns results immediately
  "modules": ["breach", "social", "darkweb", "brokers"]
}
JSON
// Response
{
  "scan_id": "scn_7f3a9d2b1e...",
  "status": "queued",
  "estimated_ms": 4200,
  "webhook_url": "https://hooks.nexus-intel.io/..."
}
RESPONSE
ParameterTypeRequiredDescription
targetstringRequiredEmail, username (with or without @), or E.164 phone number
typestringRequiredTarget type: email, username, or phone
depthstringOptionalScan depth. Defaults to standard. Deep scans include dark web (higher cost)
syncbooleanOptionalIf true, blocks until results are ready (max 30s timeout)
modulesarrayOptionalSpecific modules to run. Omit to run all available modules
GET /scan/{scan_id} Retrieve scan results
// cURL example
curl -X GET \
  https://api.nexus-intel.io/v3/scan/scn_7f3a9d2b \
  -H "Authorization: Bearer nx_live_sk_..."
